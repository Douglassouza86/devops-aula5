# Arquitetura

* As funções relacionadas ao gerenciamento das casas do jogo da velha ficarão no módulo ** jogovelha.py**

* O estado de cada casa do jogo será representadoa por uma string: "." para casa vazia; "x" para casa ocupada pelo 1°
jogador; "o" para casa ocupada pelo 2° jogador

* A função inicializar () retornará uma lista 3x, onde cada casa do jogo A função retornará  todas o estado de uma casa do jogo. A função retornara todas as casas inicialmente vazias.
